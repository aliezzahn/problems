## Getting Started

Rust library for code problems

### Prerequisites

- Rust (https://www.rust-lang.org/tools/install)

### Installation

1. Clone the repository:

   ```sh
   git clone https://github.com/aliezzahn/problems.git
   cd problems
   ```

2. Build the project:

   ```sh
   cargo build
   ```

3. Run the project:
   ```sh
   cargo run
   ```

## Running the tests

To run tests, use the following command:

```sh
cargo test
```
